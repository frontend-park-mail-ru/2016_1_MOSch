Build The Tower!
==========


Браузерная казуальная action-игра.

------
Для установки проекта

`npm install`

<br>
После установки

- для запуска тестов
`grunt test`

- для запуска статического сервера
`grunt`

- для сборки _production_-версии сервера
`grunt prod`
